# 📊 代账客户收费管理系统

> 专为代账公司设计的桌面端收费管理工具，支持客户周期收费、按次业务计费和收入统计分析。

## ✨ 功能亮点

- 🔐 **管理员登录** — 首次使用注册管理员，密码 SHA256 加密存储
- 🏠 **客户收费管理** — 年收 / 季收 / 月收客户，自动计算下一收费期和逾期状态
- 📋 **按次收费业务** — 一次性服务（工商变更、注销等），支持临时客户和多选业务类型
- 📊 **收入统计** — 月度收入汇总、年度趋势折线图、应收未收统计
- 🎨 **逾期色标** — 红/橙/黄/绿四级标注，一眼识别紧急客户
- 🔍 **搜索 + 分页** — 按姓名/电话搜索，20 条/页流畅浏览
- ✅ **批量操作** — 多选客户，一键标记收费或删除
- 📥 **Excel 导出** — 客户列表、收款记录导出为 Excel
- 📝 **操作日志** — 所有增删改查自动记录，可追溯
- 🍞 **Toast 悬浮提示** — 操作反馈优雅不打断

## 🖥️ 界面预览

主界面包含三个标签页：

| 标签页 | 功能 |
|--------|------|
| 🏠 客户收费管理 | 客户列表、收款录入、逾期监控、批量操作 |

## 🛠️ 技术栈

| 组件 | 技术 |
|------|------|
| GUI | Python tkinter（标准库，无需安装） |
| 数据库 | SQLite3（标准库，零配置） |
| 密码加密 | SHA256（hashlib 标准库） |
| Excel 导出 | openpyxl（可选，无则降级为 CSV） |
| 打包 | PyInstaller（生成独立 .exe / .app） |

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 操作系统：Windows / macOS / Linux

### 安装运行

```bash
# 1. 克隆仓库
git clone git@github.com:hilinqi/maylin.git
cd maylin

# 2. 安装依赖（可选，仅 Excel 导出需要）
pip install openpyxl

# 3. 运行
python main.py
```

首次运行需要注册管理员账号（用户名和密码自定），之后即可登录使用。

### 打包为独立应用

```bash
pip install pyinstaller
pyinstaller 代账客户收费管理系统.spec
# 生成物在 dist/ 目录下
```

## 📁 项目结构

```
├── main.py          # GUI 主程序（登录、主窗口、三个标签页）
├── database.py      # 数据库核心（建表、CRUD、统计查询）
├── requirements.txt # Python 依赖
├── .gitignore       # Git 忽略规则
├── data/            # 运行时生成，存放 clients.db 数据库文件
└── 代账客户收费管理系统.spec  # PyInstaller 打包配置
```

---

## 🗄️ 数据库设计

```
admins          — 管理员表（用户名 + 密码哈希）
clients         — 客户表（姓名、收费周期、费率、状态、收费期间）
payments        — 收款记录表（客户 ID、覆盖期间、金额、日期）
services        — 按次业务表（客户/临时客户、业务类型、收费标准、实收）
operation_logs  — 操作日志表（动作、对象、时间）
```

## 📐 使用流程

```
注册管理员 → 登录
  ├─ 添加客户（填写姓名、收费周期、金额、收费期间）
  ├─ 收款时点击「标记收费」，自动计算下一期
  ├─ 逾期客户自动红色/橙色高亮提醒
  ├─ 按次业务：录入服务 → 收款后标记「已收」
  └─ 收入统计：查看月度/年度报表，支持 Excel 导出
```
